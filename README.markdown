## Instalação

```bash
$ git clone git@github.com:gilsondev/my-vim-confs.git
$ ln -s my-vim-confs/.vim
$ ln -s my-vim-confs/.vimrc
$ ln -s my-vim-confs/.fonts.conf
```

E fica pronto para usar.
